GayHub+Hexo 搭建个人网站

To Rick Sanchez (I don’t know why, but I’m really into R&M these days)

本机环境：Windows 10, git version 2.22.0.windows.1, node version v10.16.0

技术介绍

Gayhub是什么就不介绍了，简单来说就是一个全球最大的同性交友网站（自豪
Hexo就是一个可以把你写的Markdown转为静态网页并托管给gayhub的好东西

Installations and other stuff (除特殊说明没有先后顺序之分)
  • ✨Git Bash&Node.js安装✨：官网下载最新版本并安装即可
  • ✨Github注册并新建一个仓库✨：注意仓库名需为 .github.io
  • ✨Hexo安装✨：
    使用Git Bash全局安装
    npm install -g hexo

之后我们分2个方向完成搭建

GitHub
  1. 把刚才创建的仓库Clone到本地，随便哪都行
    git clone [url] e.g. :
    $ git clone https://github.com/username/repositoryname.git
  2. 本地新建html
    在刚才clone到本地的文件夹下，新建一个index.html
  3. 代码push到github
    git add .
git commit -m [message]
git push origin main # 现在main是default
  4. 测试
    打开https://username.github.io/ 可以看到你刚才写的html网页就代表ok了
Hexo

  1. 选一个地方放你的hexo文件夹

  2. 在hexo所在文件夹下运行git bash

  3. 执行以下命令
    $ hexo init # Hexo会自动建立需要的文件
    $ npm install # 安装所需依赖包

  4. 本地打开
    $ npm install hexo-server --save # 安装server模块
    $ hexo server # 或简写为 hexo s(启动hexo服务器)

  5. 测试
    打开localhost:4000 可以看到hexo自带的页面就代表ok了

  6. 配置hexo
    修改hexo文件下的配置文件_config.yml

    deploy:
type: git
repo: <repository url> # git@github.com:XL-c137/XL-c137.github.io.git
branch: [branch] # main
message: [message] # wubbalubbadubdub, {{now('YYYY-MM-DD HH:mm:ss')}}

    保存即可，这一步是使本地hexo里的文件与.github.io建立关联；

  7. 在\hexo\public文件夹下生成静态页面
    在hexo文件夹下运行Git bash $ hexo generate # (或hexo g)

  8. 部署到github上

    $ hexo deploy # 推送到指定的库分支(配置时指定的), (或hexo d)
可能的bug及解决方案

hexo g TypeError: line.matchAll is not a function
npm ERR! code EBADPLATFORM
hexo error spawn failed 我是用第二种方法解决的，我一开始的deploy配置是用HTTPS的方式，后来又想用ssh部署，忘了把配置文件改回来
以及，注意现在main才是github的默认主分支(有点坑到眼瞎的我)
